Showing selected payment method

– paymentMethodWidgetWithFrame:

Method for retrieving payment method widget. This widget is a UIView subclass that shows currently selected payment method. State and appearance of the widget is controlled internally the only responsibility of the client is to properly show widget on screen. When user interacts with the widget the Payment Service will inform its delegate that Payment Methods View Controller should be presented on screen.

- (UIView *)paymentMethodWidgetWithFrame:(CGRect)frame

Parameters

frame

Proposed frame for widget.

Discussion

Warning: Widget will reposition its controls to reflect width changes but height is fixed to 50.
